1. Компонент должен содержать:
  1.1. Поле ввода (поиск). Placeholder: "Введите имя"
  1.2. Список вариантов для выбора 
  1.3. Кнопку "Остановить автопоиск"
  1.4. Список выбранных элементов (если пуст - скрыть)

2. Во время ввода текста, после ввода более 2 символов отправляется запрос на dadata. пример по ссылке https://dadata.ru/api/suggest/name/токен использовать token: "43b7e62f8b5e35438a657ce267979f961dcb9670"

3. После получения данных из API в списке вариантов выбора должны отобразиться доступны для выбора элементы. 
  3.1. Каждый элемент должен содержать:
    3.1.1. Имя
    3.1.2. Кнопку "Добавить"

4. При нажатии на кнопку "Добавить":
  1. Выбранный элемент должен отобразиться в списке выбранных элементов

5. При нажатии на кнопку "Остановить автопоиск":
  5.1. Текст кнопки должен измениться на "Включить автопоиск"
  5.2. При наборе текста не должен выполняться запрос к API

6. При нажатии на кнопку "Включить автопоиск":
  6.1. Текст кнопки должен измениться на "Остановить автопоиск"
  6.2. При наборе текста должен выполняться запрос к API
